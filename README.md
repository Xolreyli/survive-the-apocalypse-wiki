# Survive the Apocalypse Wiki

A GitHub Pages wiki for the Roblox game **Survive the Apocalypse**.

## How to publish
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch**
4. Branch: **main**
5. Folder: **/(root)**
6. Save — your site will publish shortly.

## Editing
- Most pages are in `/docs` as Markdown files.
- Navigation is handled by Just the Docs + page front matter.
