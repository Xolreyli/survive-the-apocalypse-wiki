# Crafting

The **Crafting Table** has **4 levels**.

> Scrap + battery costs scale based on the number of players in the lobby.

## Crafting table upgrades
- **Level 1 requires:** Scrap, Batteries  
- **Level 2 requires:** Scrap, Batteries  
- **Level 3 requires:** Scrap, Batteries, **1 Zombie Heart**  
- **Level 4 requires:** N/A *(Final level)*

## Items by crafting level
| Level | Items available |
|---:|---|
| 1 | Map, Repair Hammer, Crafting Table 2 |
| 2 | Shelf, Fence, Gate, Barbed Wire, Bear Trap, Farm Plot, Crafting Table 3 |
| 3 | Watchtower, Grenade, Molotov, Suppressor, Boost Pad, Electric Fence, Ammo Crate, Crafting Table 4 |
| 4 | Turret, Landmine, Repair Drone, Teleporter |

## Item details

| Item | Description / Use | Required resources |
|---|---|---|
| Map | Press **M** to open the map. Upgrading the generator expands the map and reveals icons (Hospital, Gun Store, Bandit Outpost, Radio Tower, Nuclear Reactor). Map also expands as players explore. | Scrap |
| Repair Hammer | Move and repair structures (fences, turrets, gates, etc). | Scrap |
| Shelf | Stores supplies. | Scrap |
| Fence | Base defense structure with health. | Scrap |
| Gate | Base access / defense. Interact with **E**. | Scrap |
| Barbed Wire | Slows zombies heavily; great for trapping hordes and setting up AoE damage with molotovs/grenades. | Scrap |
| Bear Trap | Catches zombies by the legs. Can be re-armed. | Scrap |
| Farm Plot | Grows food (3 carrots). Takes ~15 minutes to harvest. Pull carrots immediately to reset timer. | Scrap |
| Watchtower | High ground structure. Can be destroyed. | Scrap |
| Grenade | Explodes after a short delay; strong vs hordes. | Scrap + Battery |
| Molotov | Fire on ground; damage over time. | Scrap |
| Suppressor | Reduces gun noise to avoid attracting zombies. *(Not compatible with Ray Gun, Double Barrel, LMG.)* | Scrap |
| Boost Pad | Temporary speed boost. | Scrap + Battery |
| Electric Fence | Damages zombies over time while blocking. | Scrap + Battery |
| Ammo Crate | Craft ammo (costs scrap). | Scrap + Battery |
| Turret | Shoots zombies inside its green radius. Infinite ammo; can be destroyed. Placer gets the kills. | Scrap + Battery |
| Landmine | One-time explosive when a zombie steps on it. | Scrap + Battery |
| Repair Drone | Repairs nearby structures in its green radius; can be destroyed. | Scrap + Battery + Zombie Heart |
| Teleporter | Teleport between points (place 2+). Does **not** work at night. Takes ~4 seconds. Destination can be random if multiple endpoints exist. | Scrap + Battery + Zombie Heart |

## Zombie Hearts
Zombie Hearts are needed for:
- Crafting Table Level 3 upgrade
- Repair Drone
- Teleporter

Hearts drop from the **last zombie of a Blood Moon**.
